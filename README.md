# AI Resume Builder Chatbot 🤖

This project is a chatbot-based resume builder that collects user information and generates a professional resume with PDF download.

## Features
- AI-like chatbot interaction
- Step-by-step data collection
- Resume preview
- PDF download

## Tech Stack
- React.js
- Node.js
- jsPDF

## How to Run
1. Start backend:
   cd backend
   node server.js

2. Start frontend:
   cd frontend
   npm start
